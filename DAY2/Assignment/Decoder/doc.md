Decoder (Brief)

A decoder is a combinational circuit that converts n input bits into 2ⁿ output lines.

For each input combination, only one output becomes HIGH (1). It is used to identify or select a specific output based on the input code.

Example: 2-to-4 Decoder

Inputs: A, B Outputs: Y0, Y1, Y2, Y3

A B Active Output 0 0 Y0 0 1 Y1 1 0 Y2 1 1 Y3

Applications: Memory addressing, instruction decoding, and data selection.
